# Theme Alignment Story

**Goal**: Align the KPA Trainers site with the main KPA website's dark premium aesthetic.

## Background

The trainers site was built with a navy-blue/coral theme, but the main KPA website uses a sophisticated dark charcoal palette with:
- Dark background with concentric circle texture
- White monochrome logo (centered)
- Clean sans-serif typography
- White pill-shaped CTA buttons
- Minimal, premium feel

## Scope

### In Scope
1. **Color system overhaul** - Switch from navy/coral to dark charcoal palette
2. **Header redesign** - Match main KPA minimal style
3. **Typography update** - Clean sans-serif, white text
4. **Button styling** - White pill-shaped buttons
5. **Background treatment** - Subtle textured/gradient background
6. **Component updates** - Hero, TrainerCard, TrainerGrid, all profile components

### Out of Scope
- Desktop optimization (future phase)
- New features/pages
- Backend changes

## Design Reference

Main KPA site characteristics:
- Background: Dark charcoal (#1a1a1a - #2d2d2d range) with concentric circles
- Text: White/light gray
- Accents: White buttons, subtle shadows
- Logo: White monochrome, can be centered
- Overall: Dark, minimal, premium
